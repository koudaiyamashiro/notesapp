# Career Strategist AI

## 概要
求人検索ではなく、ユーザーの経験・スキル・志向性からキャリア戦略を設計するAIキャリア支援アプリ。

## 技術構成
- React
- Vite
- Tailwind CSS
- AWS Amplify
- GitHub
- Claude Code

## 現状
- LPは完成済み
- 診断画面を改善中
- 結果画面をAIダッシュボード風に改善したい
- AWS Amplifyにデプロイ済み

## 目指す体験
入力 → AI風分析 → キャリア戦略レポート表示

## 診断画面の方針
- 自由入力を減らす
- 職種選択に応じて得意領域を出し分ける
- チップ選択・プルダウン中心にする
- 「その他」の場合だけ自由入力を表示する

## 結果画面の方針
- 市場価値スコア
- レーダーチャート
- 業界適性ランキング
- 職種適性ランキング
- 企業比較
- 5年ロードマップ
- 今やるべきアクション
